# рџ“– Project Overview

OptiCrop is an AI-powered agricultural recommendation system developed to assist farmers in selecting the most suitable crop based on soil nutrients and environmental conditions.

The system analyzes:

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- Soil pH
- Rainfall

and predicts the most suitable crop for cultivation using Machine Learning algorithms.
