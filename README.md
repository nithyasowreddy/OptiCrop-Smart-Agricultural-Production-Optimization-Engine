# OptiCrop: Smart Agricultural Production Optimization Engine

## Project Description
OptiCrop is an AI-powered agricultural recommendation system that predicts the most suitable crops based on soil nutrients and environmental conditions using Machine Learning techniques.

## Features
- Crop recommendation using Machine Learning
- Soil nutrient analysis
- Environmental parameter analysis
- Interactive Flask web application
- Data visualization and prediction system

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn
- HTML/CSS/Bootstrap

## Dataset
Crop Recommendation Dataset from Kaggle.
